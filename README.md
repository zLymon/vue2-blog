# 前言

这是使用了Vue.js开发的一个个人博客

## 技术栈

vue-cli2.9.3 + vuex + vue-router + webpack + sass + axios + marked.js

## 项目运行

```
git clone https://github.com/zLymon/vue2-blog.git  

cd vue2-blog

npm install

npm run dev
```
